## MATH

### .average
calculates the average of the given values

```ruby
Math.average([0, 1, 2, 3, 4, 5])

# returns 2.5
```

It can also be used to calculate the average of weighted values

```ruby
Math.average(
  0 => 1,
  1 => 2,
  2 => 3,
  3 => 4,
  4 => 6
)

# returns 2.6666
```
