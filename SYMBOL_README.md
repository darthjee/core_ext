## Symbol

### #camelize
Camelize the symbol 'just like String#camelize'

```ruby
:underscore_sym.camelize(:upper)

# returns :UnderscoreSym
```

```ruby
:underscore_sym.camelize(:lower)

# returns :underscoreSym
```

```ruby
:underscore_sym.camelize

# returns :UnderscoreSym
```

### #underscore
underscore the symbol 'just like String#underscore'

```ruby
:symBol.underscore

# returns :sym_bol
```
